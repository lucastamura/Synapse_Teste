Criar uma pasta local


  1. Criar a pasta
  2. Abrir Cmd e entrar na pasta criada com o comando cd + url da pasta
  3. Ainda no cdm, criar um repositório local com o comando git init
  4. Criar um repositório no github
  5. Linkar o repositório local com o GitHub git remote add origin + link do repositório
